# ChronoVault

ChronoVault is a unified, privacy-friendly media tracking platform for movies, anime, games, and manga.

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: SQLite (local), PostgreSQL (production)
- ORM: Prisma

## Setup

### Frontend
cd frontend
npm install
npm run dev

### Backend
cd backend
npm install
npm run dev
